# 无约束最优化
##  最常见的使用最最多的是梯度下降法
- 优点:简单，计算量小
- 缺点：收敛速度慢，容易震动，一介收敛

设定J(x,y)求解最小值,t为表达式,t_n=(n,(x_n,y_n)

基本步骤:
- 初始位置:
- t_0 = (0,(x_0,y_0))
- t_1=1,(x_1,y_1)[x_1,y_1]^T  = [x_0,y_0]^T - lambda [J在x0,y0c处的梯度]
- 依此类推
- lambda 称为步长，也叫做学习率，是一个超参数
- 终止条件为:
  - derta [J] < sigma
  - n>N
  - 在某点的梯度值的范数<sigma
